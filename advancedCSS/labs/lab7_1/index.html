Nested Flexbox Layout

For this lab you will be incorporating nested flexbox along with media queries to practice "full-fledged" flexbox. Creating the flex container is the first step in any flexbox layout. You will be making the entire body content a flex container and including several nested flexbox within various components of the body.

In the navigation, all navigation items are aligned to the baseline of the text so they look more uniform. The login button will be positioned all the way to the right.

On the home page, the main and aside will be displayed horizontally within a tablet and desktop view, but will display vertical for mobile.
