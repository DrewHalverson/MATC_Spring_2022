Flexbox Layout

Basic lab Information

For this lab you will be incorporating flexbox into a basic web page. Refer to the Lab 8.1 specifications for complete lab requirements.

You may want to consider using Firefox for this lab as it has some very nice developer tools for working with Flexbox.

Creating the flexbox container is the first step in incorporating any flexbox layout. After defining the container, you will need to define the flex item properties.

  • For the default viewport (under 768px), the overall container will have 4 rows and 1 columns
    • Navigation section will have a width of 100% of the viewport
    • Main section will have a width of 100% of the viewport
    • Aside section will be hidden
  • For viewports 768px and greater, the overall container will still have 4 rows and 1 columns. However, a nested container should be created for the nav, main, and aside.
    • Navigation section will have a width of 20% of the viewport container
    • Aside section will displayed (formerly hidden) and have a width of 15% of the viewport container
    • Main section should automatically use any remaining space within the row
