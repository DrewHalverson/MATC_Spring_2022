Bootstrap with Customizations

For this lab you will be incorporating BootStrap v5(+) with the grid layout. This is continued practice from the previous week. You will be making the entire body content a container and including the BootStrap defined rows and columns to the various components of the body.

For this lab, we will incorporate Bootstrap's styling to include responsive design content hiding, icons, typography, and colors.

In addition, we will add some media queries to apply some changes depending on the viewport. On the home page, the main and aside will be displayed horizontally within a tablet and desktop view, but will display vertical for mobile.
